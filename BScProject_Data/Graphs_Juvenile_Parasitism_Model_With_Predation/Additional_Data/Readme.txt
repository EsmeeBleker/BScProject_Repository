This folder (Additional_Data) contains

-Readme.txt (this file)
-IEcurve.dat
  Contains i-values and associated e-values indicating the location of the first branching point in the energetic costs bifurcation for the Juvenile Parasitism Model With
  Predation, calculated using 'BPcalculations juvenile model with predation first bp energetic costs bifurcation.pdf'.
-R-naught juvenile model with predation.xlsx
  Contains tab Import_Data (values for R0 in both branching points as a function of energetic costs) and Calculations (calculations of R0 values of both branching points as
  a function of energetic costs).
